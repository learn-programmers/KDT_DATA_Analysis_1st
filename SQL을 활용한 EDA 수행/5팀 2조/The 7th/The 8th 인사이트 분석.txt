1. 처음 분석하기로 했던 8번의 프로모션 코드, 적용된 할인이 yes yes / no no로 일관된 선택지만 있어서 분석하는 의미가 없을 것 같아 주제를 변경함 -> 선호하는 결제 방식, 실제 결제 방식, 카테고리를 이용한 판매 마케팅 전략 분석

2. 첨부한 히스토그램 사진 중 pay은 원래 결제 방식(Payment Method)에 관한 그래프이고, prefer는 선호되는 결제 방식(Preferred Payment Method)임.

3. <Accessories Category> 
- 가장 선호되는 결제 방식 : Paypal (191)
- 가장 많이 사용한 실제 결제 방식 : Credit Card (215)
- PPM(소비자가 원하는 방식)에서 PM(실제 결제)값이 가장 낮아진 것 : Paypal
=> Paypal이 가장 선호되는 방식임에 불구하고, 실제론 Credit Card를 많이 사용하였음. Paypal을 선호하는 소비자 191명이 실제 결제에서 158명으로 감소된 경향으로 보아, Paypal 방식을 보완할 필요가 있음.

4. <Clothing Category> 
- 가장 선호되는 결제 방식 : Credit Card (258)
- 가장 많이 사용한 실제 결제 방식 : Credit Card (271)
- PPM(소비자가 원하는 방식)에서 PM(실제 결제)값이 가장 낮아진 것 : Cash
=> PPM 과 PM 모두 Credit Card로 나옴. 실제로 더 많이 사용하였으므로 이 방식에서는 결제 이벤트 같은 것을 결합하면 좋을 것 같음. Cash부분은 보완이 필요함. 

5. <Footwear Category> 
- 가장 선호되는 결제 방식 : Paypal (97)
- 가장 많이 사용한 실제 결제 방식 : Venmo (93)

6. <Footwear Category> 
- 가장 선호되는 결제 방식 : Debit Card (57)
- 가장 많이 사용한 실제 결제 방식 : Paypal (49)
=> Footwear, Outwear은 사용자 수가 위에 비해 많이 적어서 PPM과 PM의 차이가 적음. 결제 프로모션은 Accessories와 Clothing에 집중하는게 좋을 것 같음.
